# PDF Question Answering Agent

## Project Overview

This project implements a PDF question answering agent that extracts text from PDF documents and answers user questions based on the document content. The application combines PyMuPDF for document processing with Anthropic Claude to generate accurate, context-based responses.

The project was implemented in Google Colab using Python and demonstrates the integration of document processing with large language models for question answering.

---

## Objectives

- Read and process PDF documents
- Extract text from PDF files
- Answer questions based on document content
- Generate context-aware responses
- Integrate PyMuPDF with Anthropic Claude
- Demonstrate document-based question answering

---

---

## Project Workflow

### 1. Environment Setup

- Installed the required libraries
- Configured the Anthropic API
- Initialized the language model

### 2. PDF Processing

- Loaded the PDF document
- Extracted text from each page
- Combined the extracted text into a single document

### 3. Context Preparation

- Selected the relevant document content
- Combined the extracted text with the user's question
- Prepared the prompt for the language model

### 4. Question Answering

- Generated responses using Claude
- Answered questions using the document content
- Returned context-based responses
